## 🆘 Quick Help

**Logo not showing?**
- Make sure file is named: `al-musawir-logo.png`
- Placed in: `public/` folder
- Restart dev server: `Ctrl+C` then `npm run dev`

**Port already in use?**
```bash
npm run dev -- -p 3001
```

**Need to rebuild?**
```bash
rm -rf .next node_modules
npm install
npm run dev
```
